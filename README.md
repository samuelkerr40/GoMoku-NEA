# Go-Moku

Created for a Computer Science AQA A level Non-Exam Assessment. Completed in February 2023.

## Table of Contents
- [Controls](#Controls)
- [Rules](#Rules)

## Prerequisites
- Visual Studio 2019 or Higher
- Python 3.8 or higher
- Windows, macOS, or Linux operating system
- Only *Keyboard* can be used as input device


## Controls:
W / ⬆️ --> <sup>UP</sup>
:-
A / ⬅️ --> <sup>LEFT</sup>
S / ⬇️ --> <sup>DOWN</sup>
D / ➡️ --> <sup>RIGHT</sup>

## Rules:
1. **OBJECTIVE** Win by getting five stones in a row, any direction.
2. **VARIATE** Place one stone at a time, trying to outsmart your opponent.
3. **WIN** Be the first to get five in a row, or if the board fills up with no winner, it's a draw.
